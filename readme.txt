Morse blinker Arduino sketch with web interface for ESP8266


Tested with Adafruit HUZZAH ESP8266 board, Arduino IDE v1.8.5


To run:

1)
 Edit beac.ino to match your wifi network id and password

2)
 Build and upload to the ESP8266 board. Use Arduino IDE.

3)
 Connect to the board with a web browser. Example: "http://192.168.0.11/"
 You should see a simple web page with a form.
 In the web form, enter any text string to morse blink.
 You can monitor serial output from the board at 115200 baud to get IP address etc.


General info about the board and how to upload an Arduino sketch using Arduino IDE:
https://learn.adafruit.com/adafruit-feather-huzzah-esp8266/using-arduino-ide
